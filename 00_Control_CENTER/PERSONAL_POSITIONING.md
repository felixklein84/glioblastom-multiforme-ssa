## Public Project Quality Standard

Every public project should answer these questions clearly:

1. What problem does this project solve?
2. Why is the problem interesting?
3. What did I build?
4. What technical methods did I use?
5. What are the results or outputs?
6. What are the limitations?
7. What would I improve next?
8. Why does this project represent my abilities?

If a project cannot answer these questions, it should not be pinned publicly yet.
