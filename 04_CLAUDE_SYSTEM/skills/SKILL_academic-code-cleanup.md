# SKILL_academic-code-cleanup.md

## Purpose

Use this skill for academic, thesis-related, mathematical, scientific, or research code.

## Priorities

1. Reproducibility
2. Model clarity
3. Mathematical assumptions
4. Clean folder structure
5. Separation of code, notebooks, data, and outputs
6. Clear README
7. Privacy and data safety
8. Honest limitations
9. Example plots or results
10. Link to thesis context where appropriate

## Recommended Structure

```text
project/
├── README.md
├── CLAUDE.md
├── src/
├── notebooks/
├── examples/
├── data/
│   └── README.md
├── outputs/
├── tests/
├── docs/
└── requirements.txt or pyproject.toml
```
