# SKILL_portfolio-project-upgrader.md

## Purpose

Use this skill when transforming a repository into a professional portfolio project.

## Core Workflow

1. Read CLAUDE.md.
2. Read README.md.
3. Inspect repository structure.
4. Identify project purpose.
5. Identify public positioning.
6. Plan small changes.
7. Improve documentation.
8. Improve reproducibility.
9. Add or plan visual assets.
10. Review privacy risks.
11. Suggest GitHub description, topics, and portfolio summary.

## Output Format

```markdown
# Portfolio Upgrade Plan

## Public Positioning

## README Improvements

## Reproducibility Improvements

## Visual Asset Plan

## Code / Structure Improvements

## Privacy Risks

## GitHub Description

## Suggested GitHub Topics

## Portfolio Case Study Summary

## Next Implementation Step
```
