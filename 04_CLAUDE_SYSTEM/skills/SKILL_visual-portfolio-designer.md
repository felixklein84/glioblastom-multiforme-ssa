# SKILL_visual-portfolio-designer.md

## Purpose

Use this skill when planning visuals for GitHub repositories, portfolio case studies, README files, screenshots, diagrams, thumbnails, or project previews.

## Recommended Visual Types

- simulation GIF
- architecture diagram
- state transition diagram
- workflow diagram
- dashboard screenshot
- example output plot
- project thumbnail
- before/after comparison
- data pipeline diagram

```markdown
# Visual Portfolio Plan

## Project Visual Goal

## Required Visuals

## Nice-to-Have Visuals

## README Visual Placement

## Portfolio Thumbnail Concept

## Diagram Concepts

## Screenshot Checklist

## File Naming Plan

## Asset Folder Structure

```
